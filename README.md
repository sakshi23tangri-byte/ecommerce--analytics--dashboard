# E-Commerce Analytics Dashboard Project

## Project Overview
Built an end-to-end e-commerce analytics system using PostgreSQL, SQL window functions, CTEs, and Power BI.

## Tools Used
- PostgreSQL
- pgAdmin 4
- SQL
- Power BI

## Key Analysis Performed
- Total Revenue
- Order Status Analysis
- Monthly Revenue Trend
- Top Customers
- Best Selling Products
- Customer Segmentation
- Revenue Growth using LAG()
